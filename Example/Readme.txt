Using documents generated by BarnackTex requires a browser with enabled Cross-Origin Resource Sharing (CORS).

To enable CORS on Firefox:
 - Open Firefox, and on the address bar type "about:config".
 - Search for "security.fileuri.strict_origin_policy".
 - Toggle the value from TRUE to FALSE.
 - Close the browser and launch it again.
